HICC 2024년 2학기 프로젝트 대회 프론트엔드용 과제입니다.

페이지 스크린샷과 과제 폴더를 참조하며, HTML과 CSS를 사용해 뉴스 기사 웹페이지를 만들어주세요.

## 참조 사항

- 페이지 내에서 사용된 텍스트와 이미지는 과제 폴더에 포함되어 있습니다.
- 페이지 세부 디자인과 수치가 완전히 동일할 필요는 없습니다. 레이아웃과 전체적인 느낌만 유지하며 자유롭게 만들어 주세요.
- 페이지를 꼭 전부 완성할 필요는 없습니다. 할 수 있는 만큼만 진행하셔도 됩니다.

## 요구 사항

- HTML 5 이상의 웹 문서로 제작하여 주세요.
- HTML과 CSS, JS 파일을 별도 파일로 분리하여 주세요.
- 시멘틱 태그를 적극 활용해 주세요.
- 레이아웃을 구성할 때애는 CSS Flexbox를 사용하여 주세요.
- 길이 단위는 rem을 사용하여 주세요.
- 서체는 Pretendard 서체를 웹 폰트로 적용하여 주세요.
    - https://github.com/orioncactus/pretendard

## 제출 방법

완성된 과제는 깃허브 리포지토리를 만들어 제출합니다. 자신의 개인 계정에 모든 구성 요소 파일이 포함된 리포지토리를 생성한 후 리포지토리 링크를 카카오톡 채팅방으로 제출해주세요.
